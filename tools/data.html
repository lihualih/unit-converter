About Data Storage Conversion

Data storage units follow the binary system where each unit is 1024 times the previous one (2^10). This is different from the metric system where each unit is 1000 times the previous.

Data Units

  • Byte (B): The basic unit. One byte stores one character.
  • Kilobyte (KB): 1 KB = 1,024 bytes. A short text document.
  • Megabyte (MB): 1 MB = 1,024 KB = 1,048,576 bytes. A photo or short song.
  • Gigabyte (GB): 1 GB = 1,024 MB. A movie or several hundred photos.
  • Terabyte (TB): 1 TB = 1,024 GB. A large hard drive capacity.

Note on Binary vs Decimal

Storage manufacturers often use decimal (1 KB = 1000 bytes) while operating systems use binary (1 KB = 1024 bytes). This converter uses the binary (IEC) standard. The IEC also defines KiB, MiB, GiB, TiB for binary units.
