# 🔄 Unit & Currency Converter — Free Online Conversion Tools

Free online unit and currency converter supporting 9+ conversion types. Real-time currency rates, instant conversion.

**GitHub**: https://github.com/lihualih/unit-converter

---

## 🛠️ Converters Included

| Converter | Supported Units |
|-----------|----------------|
| **Currency** | USD, EUR, GBP, JPY, CNY, + more (live rates) |
| **Length** | m, km, mi, ft, in, cm, mm, yd |
| **Weight** | kg, g, lb, oz, ton, st |
| **Temperature** | °C, °F, K |
| **Speed** | km/h, mph, m/s, knots |
| **Area** | sq m, sq ft, sq km, acres, hectares |
| **Volume** | liters, gallons, ml, cups, fl oz |
| **Data Storage** | bytes, KB, MB, GB, TB |
| **Time** | seconds, minutes, hours, days, weeks, years |

## ✨ Features

- ✅ Real-time conversion (as you type)
- ✅ Live currency exchange rates
- ✅ Swap button to reverse conversion
- ✅ Mobile responsive
- ✅ Dark mode
- ✅ SEO optimized (each converter has its own page)

## ⚡ Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- **Open Exchange Rates API** — Free currency rates
- Zero dependencies, instant loading

## 🚀 How to Use

1. Select the converter you need
2. Enter a value
3. Select source and target units
4. See instant results
5. Copy the converted value

## 📦 Deployment

```bash
npm i -g vercel
vercel login
vercel --prod
```
